## GitHub Pages の有効化手順

1. リポジトリの **Settings** タブを開く
2. 左メニューの **Pages** を選択
3. **Source** を `Deploy from a branch` に設定
4. **Branch** を `main` / `/ (root)` に設定して **Save**
5. 数分後に公開URLが発行される
